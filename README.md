# Blog Application

A blog application built with React, TypeScript, and Firebase.

## Features

- User Authentication
- Add Blog
- Edit Blog
- Delete Blog
- View Blogs
- Pagination
- Firestore Database
- Form Validation using Zod
- React Hook Form

## Tech Stack

- React
- TypeScript
- Vite
- Firebase Authentication
- Cloud Firestore
- React Router
- React Hook Form
- Zod
